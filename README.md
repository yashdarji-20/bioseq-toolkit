## Overview

BioSeq Toolkit is a command-line and library toolkit for common bioinformatics
sequence-analysis tasks — validation, GC content, transcription, translation,
FASTA parsing, and more — implemented from scratch in pure Python with a clean,
tested, modular architecture.

## Installation

```bash
git clone https://github.com/yashdarji-20/bioseq-toolkit.git
cd bioseq-toolkit
python -m venv .venv
.venv\Scripts\activate
pip install -e ".[dev]"
```

## Usage

```bash
bioseq gc data/sample.fasta
```
